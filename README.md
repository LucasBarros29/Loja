# Loja.cpp

## Descrição
Sistema de gerenciamento de estoque de instrumentos musicais desenvolvido em C++, permitindo cadastrar, listar, atualizar e remover produtos, além de simular vendas e salvar os dados em arquivos.

## Conceitos Utilizados
- Structs
- Vetores
- Funções
- Manipulação de arquivos
- Organização modular
- Estruturas de dados básicas em C++

## Objetivo
Aplicar conceitos fundamentais de algoritmos e programação na criação de um sistema funcional de controle de estoque em ambiente console.
